The hX+jF Demo

Stellar Help Exchange (hX) is an equitable cloud-based marketplace. The demo runs on Stellar testnet in 3 steps:
  • reset hX on testnet;
  • setup hX demo data; and
  • run the demo.
Each step has one or more jobs you request from this browser. Cloudflare Job Fair (jF) matches your job requests with Job Agents that offer the jobs you need. Without the jF functionality this demo would not run, so it is actually a double demo, hX+jF. All jF messages are signed by the sender and verified on the edge and by the recipient. It is worth noting here that jF Job Agents in qa MUST use mTLS. On the edge, each jF participant - a Job Agent or a Job Client like this browser - is represented by an open websocket connection and the associated attachment. The attachment keeps the breakpoint of the State and Context objects for the connection.
