The TM Test

Stellar Testnet Monitor (TM) is used to start Job Agents whenever the demo is requested. If another demo is in progress, the request waits in the queue. The implementation is based on the Stellar TESTNET orderbook. The TM agent places a sell order for the Monitor Asset (MA). You place a buy order for the same amount of MA. When your order executes, the demo starts for you. When the demo completes (or when the timeout on the agent expires), another sell order for the same amount of MA is being placed - either by you or by the agent. Now the TM is either waiting for someone else to place a buy order, or executes an order already placed. Basically, the TM is a cloud-based semaphore.

This page tests the semaphore's usage. The hX+jF demo page is based on this page.
